Processing commandsedit

ESQL processing commands change an input table by adding, removing, or changing rows and columns.

ESQL supports these processing commands:

  • DISSECT
  • DROP
  • EVAL
  • GROK
  • LIMIT
  • PROJECT
  • RENAME
  • SORT
  • STATS ... BY
  • WHERE
